# AI-Powered Healthcare Incident Management System

## Project Overview
This ServiceNow application monitors patient health records and automatically creates healthcare incidents when a patient's risk level becomes Critical.

## Features
- Patient Management
- Health Record Management
- Automated Critical Health Alert Flow
- Automatic Healthcare Incident Creation
- ServiceNow App Engine Studio Implementation
